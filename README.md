# QuickLead Web App

QuickLead is an AI-powered leadership web application that provides instant summaries and curated resources for leadership topics.

## Features
- AI-powered summaries
- URL filtering for safe browsing
- Personalized recommendations
- FastAPI backend for summarization

## Setup Instructions
1. Clone the repository
2. Install dependencies: `npm install` for frontend, `pip install fastapi uvicorn` for backend
3. Configure Supabase and environment variables
4. Run backend: `uvicorn app:app --reload`
5. Run frontend: `npm run dev`
