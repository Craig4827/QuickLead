# QuickLead Web App

QuickLead is an AI-powered leadership web application that provides instant summaries and curated resources for leadership topics.

## Features
- AI-powered summaries
- URL filtering for safe browsing
- Personalized recommendations

## Setup Instructions
1. Clone the repository
2. Install dependencies: `npm install`
3. Configure Supabase and environment variables
4. Deploy Supabase Edge Function: `quicklead-search`
5. Run the app: `npm run dev`
